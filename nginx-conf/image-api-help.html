This service scales/crops images with fixed configurable resolutions and provides metadata.

Endpoints

  • /image/(resize|crop)/{WIDTH}x{HEIGHT}/{IMAGE}
    Provides a resized/cropped version of IMAGE.
  • /image/info/{IMAGE}
    Provides image type and resolution of IMAGE as JSON.
  • /image/info
    Lists available image resolutions and transformations as JSON.
